# RHFNumberInput<sup><img src={NewImg} alt="NEW" height="40" /></sup>

<AvailabilityBanner componentName="RHFNumberInput" version="2.1" />

**RHFNumberInput** is an enhanced version of the [TextField](https://mui.com/material-ui/react-text-field/)
component which accepts only **integer** or **float** values. It retains
the core functionality, styling options, and flexibility of the TextField,
while seamlessly integrating with react-hook-form for number-specific input handling.

## Usage

```js
import RHFNumberInput, { RHFNumberInputProps } from '@nish1896/rhf-mui-components/mui/number-input';
```

To render a fully functional `RHFNumberInput`, you only need 3-4 essential props.

```jsx
<RHFNumberInput
  fieldName="age"
  control={control}
  errorMessage={errors?.age?.message}
/>
```
