# Mui Components

The components in this module are based on the core [Material-UI Input Components](https://v5.mui.com/material-ui/all-components/#inputs),
such as [TextField](https://v5.mui.com/material-ui/react-text-field/) and
[Select](https://v5.mui.com/material-ui/react-select/). They share similar props 
and offer comparable customization options.
