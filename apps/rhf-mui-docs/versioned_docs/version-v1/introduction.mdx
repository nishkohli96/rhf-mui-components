# Introduction
Easily design forms in minutes using the popular [react-hook-form](https://react-hook-form.com/)
library alongside [Material-UI](https://v5.mui.com/material-ui/getting-started/).
This package offers a suite of 15 reusable and customizable MUI components,
reducing code duplication and providing the flexibility to style form fields 
according to your website's design. 

You can also find some additional components in this package like **color-picker** and 
**rich text editor** that are not currently a part of Material-UI,
but will come in handy when you need them. More components will be added in the future based on the feedback from the 
community. You are most welcome to raise a feature request or suggest 
bug fix by contacting the author of this package. 


## Installation

:::caution
**Deprecation of Version 1**<br/>
It is highly recommended to upgrade to the latest version of both [Material UI](https://mui.com/material-ui/)
and this package to access the newest features.

If your project is currently using **MUI v5**, please follow the upgrade path below:

1. First, upgrade this package from v1 → v2 by following the [v2 migration guide](../version-v2/migration.md).
2. After addressing the breaking changes introduced in `v2`, upgrade from **v2** → **v3** using the
[v3 migration guide](../version-v3/migration.md).
:::

To get started, install [react-hook-form](https://www.npmjs.com/package/react-hook-form) and [@nish1896/rhf-mui-components](https://www.npmjs.com/package/@nish1896/rhf-mui-components/v/1.0.3).

```
npm i react-hook-form @nish1896/rhf-mui-components@1.0.3
```
```
yarn add react-hook-form @nish1896/rhf-mui-components@1.0.3
```

Ensure that you also have installed [Material UI](https://www.npmjs.com/package/@mui/material). 
If not, refer to the [installation guide](https://v5.mui.com/material-ui/getting-started/installation/). 
For datepickers, follow the [installation guide](https://v6.mui.com/x/react-date-pickers/getting-started/) 
to install [@mui/x-date-pickers](https://www.npmjs.com/package/@mui/x-date-pickers).
The form components are compatible with both [v5](https://v5.mui.com/material-ui/getting-started/) 
and [v6](https://v6.mui.com/material-ui/getting-started/) of material-ui.

**Optional** : If you are using a third party library for form validation, say [joi](https://www.npmjs.com/package/joi), make
sure to install its npm package along with the [@hookform/resolvers](https://www.npmjs.com/package/@hookform/resolvers) package.

```
npm i @hookform/resolvers joi
```
```
yarn add @hookform/resolvers joi
```
