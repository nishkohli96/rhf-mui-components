# RHFSlider

Sliders reflect a range of values along a bar, from which users may select a 
single value. They are ideal for adjusting settings such as volume or brightness.
`RHFSlider` extends Mui [Slider](https://mui.com/material-ui/react-slider/),
so it accepts almost all props of `Slider` and can be customized accordingly!

### Usage

```js
import { RHFSlider, RHFSliderProps } from "@nish1896/rhf-mui-components";
```

```jsx
<RHFSlider
  fieldName="score"
  register={register}
  defaultValue={40}
  errorMessage={errors?.score?.message}
/>
```

Please visit the below links to check the component usage.

- [RHFSlider Example](https://mui.com/material-ui/react-Slider/)
- Mui [Slider](https://mui.com/material-ui/react-Slider/) Component
