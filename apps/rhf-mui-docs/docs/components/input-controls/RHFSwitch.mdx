# RHFSwitch

Switches toggle the state of a single setting on or off. `RHFSwitch` extends the [Switch](https://mui.com/material-ui/react-switch/)
component and can be customized in a similar way.

### Usage

```js
import { RHFSwitch, RHFSwitchProps } from '@nish1896/rhf-mui-components';
```

```jsx
<RHFSwitch
  fieldName="darkTheme"
  control={control}
  label="Enable Dark Theme?"
/>
```
