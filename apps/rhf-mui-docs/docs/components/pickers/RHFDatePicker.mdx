# RHFDatePicker

`RHFDatePicker` extends the [DatePicker](https://mui.com/x/react-date-pickers/date-picker/)
component to let users select a date and can be customized in a similar way.

:::info
`RHFDatePicker` uses [moment](https://momentjs.com/docs/) as its default adapter,
returning a `moment` object. To switch to a different date adapter, please refer 
the [customization guide](../../customization).
:::

### Usage

```js
import { RHFDatePicker, RHFDatePickerProps } from '@nish1896/rhf-mui-components';
```

```jsx
<RHFDatePicker
  fieldName="dateOfBirth"
  register={register}
  setValue={setValue}
  errorMessage={errors?.dateOfBirth?.message}
/>
```
