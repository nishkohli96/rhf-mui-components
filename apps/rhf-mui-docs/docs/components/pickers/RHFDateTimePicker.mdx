# RHFDateTimePicker

`RHFDateTimePicker` extends [DateTimePicker](https://mui.com/x/react-date-pickers/date-time-picker/)
component, allowing users to select the date and time for an event.

:::info
`RHFDateTimePicker` uses [moment](https://momentjs.com/docs/) as its default adapter,
returning a `moment` object. To switch to a different date adapter, please refer 
the [customization guide](../../customization).
:::

### Usage

```js
import { RHFDateTimePicker, RHFDateTimePickerProps } from '@nish1896/rhf-mui-components';
```

```jsx
<RHFDateTimePicker
  fieldName="departureOfFlight"
  register={register}
  setValue={setValue}
  errorMessage={errors?.departureOfFlight?.message}
/>
```
