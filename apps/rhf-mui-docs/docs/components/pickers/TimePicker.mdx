# RHFTimePicker

Switches toggle the state of a single setting on or off. `RHFTimePicker` extends Mui [Switch](https://mui.com/material-ui/react-switch/),
so it accepts almost all props of `Switch` and can be customized accordingly!

### Usage

```js
import { RHFTimePicker, RHFTimePickerProps } from "@nish1896/rhf-mui-components";
```

```jsx
<RHFTimePicker
  fieldName="dob"
  register={register}
  setValue={setValue}
  errorMessage={errors?.dob?.message}
/>
```

Please visit the below links to check the component usage.

- [RHFTimePicker Example](https://mui.com/material-ui/react-switch/)
- Mui [Switch](https://mui.com/material-ui/react-switch/) Component
