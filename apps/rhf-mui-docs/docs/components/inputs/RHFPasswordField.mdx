# RHFPasswordField

`RHFPasswordField` is a customized version of [TextField](https://mui.com/material-ui/react-text-field/),
which as the name suggests, will be mainly used to input password on signup and login pages. By using this 
component in your form, users will be able to toggle password visibility while filling up their credentials.

### Usage

```js
import { RHFPasswordField, RHFPasswordFieldProps } from '@nish1896/rhf-mui-components';
```

`RHFPasswordField` also requires a bare minimum of 3-4 props to render a fully functional form-field.

```jsx
<RHFPasswordField
  fieldName="password"
  register={register}
  errorMessage={errors?.password?.message}
/>
```
