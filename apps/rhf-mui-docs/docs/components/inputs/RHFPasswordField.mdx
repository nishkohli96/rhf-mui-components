# RHFPasswordField

`RHFPasswordField` an enhanced version of the [TextField](https://mui.com/material-ui/react-text-field/) component,
designed specifically for password input on signup and login pages. 
This component allows users to toggle password visibility, 
offering a more user-friendly experience when entering credentials.

### Usage

```js
import { RHFPasswordField, RHFPasswordFieldProps } from '@nish1896/rhf-mui-components';
```

`RHFPasswordField` also requires  just 3-4 essential props to render a fully functional field.

```jsx
<RHFPasswordField
  fieldName="password"
  register={register}
  errorMessage={errors?.password?.message}
/>
```
