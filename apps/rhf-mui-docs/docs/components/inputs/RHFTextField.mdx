# RHFTextField

Text fields are a core component of any form, allowing users to input text into the UI.
The `RHFTextField` component extends the functionality of [TextField](https://mui.com/material-ui/react-text-field/),
thus accepting almost all TextField props while providing additional customization options!

### Usage

```js
import { RHFTextField, RHFTextFieldProps } from '@nish1896/rhf-mui-components';
```

To render a fully functional `RHFTextField`, you only need 3-4 essential props.

```jsx
<RHFTextField
  fieldName="firstName"
  register={register}
  errorMessage={errors?.firstName?.message}
/>
```
