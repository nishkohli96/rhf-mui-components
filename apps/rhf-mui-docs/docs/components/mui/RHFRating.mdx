# RHFRating

Ratings provide insight into opinions and experiences, and they can
allow users to submit a rating of their own. **RHFRating** extends the
[Rating](https://v7.mui.com/material-ui/react-rating/) component,
so it accepts almost all `Rating` props and can be customized in the same way.

## Usage

```js
import RHFRating, { RHFRatingProps } from '@nish1896/rhf-mui-components/mui/rating';
```

```jsx
<RHFRating
  fieldName="rating"
  control={control}
/>
```
