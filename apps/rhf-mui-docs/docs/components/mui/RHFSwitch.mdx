# RHFSwitch

Switches toggle the state of a single setting on or off. **RHFSwitch**
extends the [Switch](https://v7.mui.com/material-ui/react-switch/) component
and can be customized in a similar way.

## Usage

```js
import RHFSwitch, { RHFSwitchProps } from '@nish1896/rhf-mui-components/mui/switch';
```

```jsx
<RHFSwitch
  fieldName="darkTheme"
  control={control}
  label="Enable Dark Theme?"
/>
```
