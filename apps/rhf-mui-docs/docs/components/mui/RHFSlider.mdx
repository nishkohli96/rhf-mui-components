# RHFSlider

Sliders display a range of values along a bar, from which users may select a
single value. They are ideal for adjusting settings such as volume or brightness.
**RHFSlider** extends [Slider](https://v7.mui.com/material-ui/react-slider/)
and can be customized in a similar way.

## Usage

```js
import RHFSlider, { RHFSliderProps } from '@nish1896/rhf-mui-components/mui/slider';
```

```jsx
<RHFSlider
  fieldName="score"
  register={register}
/>
```
