# RHFMultiAutocomplete

**RHFMultiAutocomplete** is an enhanced version of [RHFSelect](./RHFSelect.mdx),
designed for efficient handling of large datasets. It supports selecting one, multiple,
or even all options and offers functionality similar to `RHFSelect`.
This component is ideal for use cases requiring dynamic filtering or managing
extensive options seamlessly.

### Usage

```js
import RHFMultiAutocomplete, { RHFMultiAutocompleteProps } from '@nish1896/rhf-mui-components/mui/multi-autocomplete';
```

```jsx
<RHFMultiAutocomplete
  fieldName="countriesVisited"
  control={control}
  options={['Azerbaijan', 'Canada', 'India']}
  errorMessage={errors?.countriesVisited?.message}
/>
```

For options as an array of objects:

```jsx
<RHFMultiAutocomplete
  fieldName="countriesVisited"
  control={control}
  options={countryList} // imported from RHFCountrySelect
  labelKey="name"
  valueKey="name"
  errorMessage={errors?.countriesVisited?.message}
/>
```

:::note
When using an array of objects for options, both `labelKey` and `valueKey` 
are required. If either is missing, an error will be thrown.
:::
