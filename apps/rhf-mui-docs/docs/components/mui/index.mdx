# MUI Components

The components in this module are based on core [Material UI input components](https://v7.mui.com/material-ui/all-components/#inputs),
such as [TextField](https://v7.mui.com/material-ui/react-text-field/) and
[Select](https://v7.mui.com/material-ui/react-select/). They share similar props
and offer comparable customization options.
