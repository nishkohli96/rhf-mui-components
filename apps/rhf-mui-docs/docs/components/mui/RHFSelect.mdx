# RHFSelect

**RHFSelect** is an extended version of [Select](https://v7.mui.com/material-ui/react-select/),
allowing users to select one or more options. One key advantage is its
ability to _**accept both arrays and arrays of objects**_, making it easier to
render options with differing labels and values without custom logic.

:::tip
If the number of options **exceeds 20**, consider using [RHFAutocomplete](./RHFAutocomplete.mdx)
or [RHFMultiAutocomplete](./RHFMultiAutocomplete.mdx) for better searchability, improved keyboard
navigation, and overall performance.
:::

## Usage

```js
import RHFSelect, { RHFSelectProps } from '@nish1896/rhf-mui-components/mui/select';
```

Here is a basic example with an array of strings:

```jsx
<RHFSelect
  fieldName="color"
  control={control}
  options={['Red', 'Blue', 'Orange']}
/>
```

For options as an array of objects:

```jsx
<RHFSelect
  fieldName="country"
  control={control}
  options={[
    { code: 'AUS', country: 'Australia' },
    { code: 'IN', country: 'India' },
    { code: 'UAE', country: 'United Arab Emirates' },
  ]}
  labelKey="country"
  valueKey="code"
/>
```

:::warning
When using an array of objects for options, both `labelKey` and `valueKey` 
are required. If either is missing, an error will be thrown.
:::

You can extend or reuse the `RHFSelect` component by creating your own wrapper component.
