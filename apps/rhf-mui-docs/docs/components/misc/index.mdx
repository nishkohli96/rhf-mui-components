# Miscellaneous Components

**These components are not part of the [Material UI component list](https://v7.mui.com/material-ui/all-components/)
and are derived from external components published on [npm](https://www.npmjs.com/).**

You need to install the corresponding npm package separately for each derived
component, or create your own component if the existing implementation does not
meet your requirements.

Suggestions for improving existing implementations are welcome.
Feel free to open a PR with your implementation of a missing component to help
other developers as well.

:::warning
Use the default import for each component in this module to avoid
"Package NOT FOUND" errors at runtime.

Please thoroughly read the documentation for the relevant package, including
the props it requires.
:::

Alternative packages with similar functionality are suggested for developer
convenience. However, you will need to create your own implementation to
integrate those alternatives.

As always, each component demo includes a link to the relevant source code.
