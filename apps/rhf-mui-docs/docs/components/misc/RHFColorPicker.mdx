# RHFColorPicker

**RHFColorPicker** is derived from the [ColorPicker](https://www.npmjs.com/package/react-color-palette#usage) 
component of [react-color-palette](https://www.npmjs.com/package/react-color-palette) package.

```
npm install react-color-palette
```

```
yarn add react-color-palette
```

## Usage

```js
import RHFColorPicker, { RHFColorPickerProps } from '@nish1896/rhf-mui-components/misc/color-picker';
```

The color-picker accepts string value in `hex`, `rgb`, `hsl` or text format, such as:
- `#00ffff`
- `rgb(0 255 255)`
- `hsl(180 100% 100% / 1)`
- `cyan`

If an invalid value is provided, the default color `#000000`(black) will be selected.

```jsx
<RHFColorPicker
  fieldName='favouriteColor'
  value={getValues('favouriteColor')}
  onValueChange={color => setValue('favouriteColor', color.hex)}
/>
```

:::warning
The `onValueChange` prop is mandatory and allows you to choose the format
of the selected color value based on your preference. The returned value
need to be passed to the `setValue` function to update the form state.
:::

The returned value from `onValueChange` is an object with the equivalent
hex, rgb and hsv values.

```js
{
  hex: '#00ffff',
  hsv: {
    a: 1,
    h: 180,
    s: 100,
    v: 100
  },
  rgb: {
    a: 1,
    b: 255,
    g: 255,
    r: 0
  }
}
```
