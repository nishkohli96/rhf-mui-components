# RHFColorPicker

**RHFColorPicker** is derived from the [ColorPicker](https://www.npmjs.com/package/react-color-palette#usage)
component from the [react-color-palette](https://www.npmjs.com/package/react-color-palette) package.

```
npm install react-color-palette
```

```
yarn add react-color-palette
```

## Usage

```js
import RHFColorPicker, { RHFColorPickerProps } from '@nish1896/rhf-mui-components/misc/color-picker';
```

The color picker accepts string values in `hex`, `rgb`, `hsl`, or text format, such as:
- `#00ffff`
- `rgb(0 255 255)`
- `hsl(180 100% 100% / 1)`
- `cyan`

If the field value is uninitialized, the default color **black** (`#000000`) is selected,
which can be configured using the `defaultColor` prop. Refer to the
[colorToString](../../form-helpers/colorToString.md) function to convert
an **RGB** or **HSV** color object into a string, which can be used to
set the default color for the color picker.

```jsx
<RHFColorPicker
  fieldName="favouriteColor"
  control={control}
/>
```

The color value returned from the `onValueChange` prop is an object containing the equivalent
**hex**, **rgb**, and **hsv** values.

```js
{
  hex: '#00ffff',
  hsv: {
    a: 1,
    h: 180,
    s: 100,
    v: 100
  },
  rgb: {
    a: 1,
    b: 255,
    g: 255,
    r: 0
  }
}
```
