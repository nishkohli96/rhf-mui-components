# RHFRating

Ratings provide insight regarding others' opinions and experiences, and can
allow the user to submit a rating of their own. `RHFRating` extends Mui 
[Rating](https://mui.com/material-ui/react-rating/),
so it accepts almost all props of `Rating` and can be customized accordingly!

### Usage

```js
import { RHFRating, RHFRatingProps } from "@nish1896/rhf-mui-components";
```

```jsx
<RHFRating
  fieldName="rating"
  control={control}
  errorMessage={errors?.rating?.message}
/>
```

Please visit the below links to check the component usage.

- [RHFRating Example](https://mui.com/material-ui/react-Rating/)
- Mui [Rating](https://mui.com/material-ui/react-rating/) Component
