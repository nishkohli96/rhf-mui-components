# RHFCheckbox

The `RHFCheckbox` component renders a checkbox, allowing users to toggle options on or off, 
such as subscribing to newsletters. It extends the [Checkbox](https://mui.com/material-ui/react-checkbox/)
component from MUI, accepting most of its props and offering similar 
customization options.

### Usage

```js
import { RHFCheckbox, RHFCheckboxProps } from '@nish1896/rhf-mui-components';
```

```jsx
<RHFCheckbox
  fieldName="agreeTnC"
  control={control}
  label="Agree to Terms & Conditions"
  errorMessage={errors?.agreeTnC?.message}
/>
```
