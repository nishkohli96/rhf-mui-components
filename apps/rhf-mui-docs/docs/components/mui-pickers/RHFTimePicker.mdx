# RHFTimePicker

**RHFTimePicker** extends the [TimePicker](https://v8.mui.com/x/react-date-pickers/time-picker/)
component to allow users to select a time for an event.

Starting with **version 4**, this module also exports additional variant
components like `RHFDesktopTimePicker`, `RHFMobileTimePicker`, and
`RHFStaticTimePicker`.

## Usage

```ts
import {
  RHFTimePicker,
  RHFDesktopTimePicker,
  RHFMobileTimePicker,
  RHFStaticTimePicker,
  type RHFTimePickerProps,
  type RHFDesktopTimePickerProps,
  type RHFMobileTimePickerProps,
  type RHFStaticTimePickerProps
} from '@nish1896/rhf-mui-components/mui-pickers/time';
```

```jsx
<RHFTimePicker
  fieldName="wakeUpTime"
  control={control}
/>
```
