# RHFTimePicker

**RHFTimePicker** extends the [TimePicker](https://mui.com/x/react-date-pickers/time-picker/)
component to allow users to select time for an event.

### Usage

```js
import RHFTimePicker, { RHFTimePickerProps } from '@nish1896/rhf-mui-components/mui-pickers/time';
```

```jsx
<RHFTimePicker
  fieldName="wakeUpTime"
  register={register}
  setValue={setValue}
  errorMessage={errors?.wakeUpTime?.message}
/>
```
