# RHFDatePicker

`RHFDatePicker` extends the [DatePicker](https://mui.com/x/react-date-pickers/date-picker/)
component to let users select a date and can be customized in a similar way.

### Usage

```js
import RHFDatePicker, { RHFDatePickerProps } from '@nish1896/rhf-mui-components/mui-pickers/date';
```

```jsx
<RHFDatePicker
  fieldName="dateOfBirth"
  register={register}
  setValue={setValue}
  errorMessage={errors?.dateOfBirth?.message}
/>
```
