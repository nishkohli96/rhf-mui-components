# RHFDateTimePicker

**RHFDateTimePicker** extends the [DateTimePicker](https://v8.mui.com/x/react-date-pickers/date-time-picker/)
component, allowing users to select a date and time for an event.

Starting with **version 4**, this module also exports `RHFDesktopDateTimePicker`,
`RHFMobileDateTimePicker`, and `RHFStaticDateTimePicker`.

## Usage

```ts
import { RHFDateTimePicker, type RHFDateTimePickerProps } from '@nish1896/rhf-mui-components/mui-pickers/date-time';
```

```jsx
<RHFDateTimePicker
  fieldName="departureOfFlight"
  control={control}
/>
```
