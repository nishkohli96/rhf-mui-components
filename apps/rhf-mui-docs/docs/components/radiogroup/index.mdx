# RHFRadioGroup

`RHFRadioGroup` component renders a checkbox which allows the user to turn an option on or off, for 
example, to subscribe to promotional emails. `RHFRadioGroup` extends Mui [Checkbox](https://mui.com/material-ui/react-checkbox/),
so it accepts almost all props of `Checkbox` and can be customized in a similar way!

### Usage

```js
import { RHFRadioGroup, RHFRadioGroupProps } from "@nish1896/rhf-mui-components";
```

```jsx
<RHFRadioGroup
  fieldName="favouriteColor"
  control={control}
  options={["Red", "Blue", "Orange"]}
  errorMsg={errors?.favouriteColor?.message}
/>

/* With an array of objects as options */
<RHFRadioGroup
  fieldName="nationality"
  control={control}
  options={[
    { code: "AUS", country: "Australia" },
    { code: "UAE", country: "United Arab Emirates" },
    { code: "IN", country: "India" },
  ]}
  labelKey="country"
  valueKey="code"
  errorMsg={errors?.nationality?.message}
/>
```
