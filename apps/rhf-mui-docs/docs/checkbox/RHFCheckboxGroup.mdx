# RHFCheckboxGroup

`RHFCheckboxGroup` component renders multiple checkboxes which allows the user
to select one or more items from a set. `RHFCheckboxGroup` accepts options as
array or an array of objects and the [checkbox](https://mui.com/material-ui/react-checkbox/)
corresponding to each option can be customized!

### Usage

```js
import { RHFCheckboxGroup, RHFCheckboxGroupProps } from "@nish1896/rhf-mui-components";
```

```jsx
<RHFCheckboxGroup
  fieldName="color"
  control={control}
  label="Agree to Terms & Conditions"
	options={["Red", "Blue", "Orange"]}
  errorMsg={errors?.color?.message}
/>

/* With an array of objects as options */
<RHFCheckboxGroup
  fieldName="countries"
  control={control}
  label="Select Countries You've visited"
  options={[
    { code: "AUS", country: "Australia" },
    { code: "UAE", country: "United Arab Emirates" },
    { code: "IN", country: "India" },
  ]}
  labelKey="country"
  valueKey="code"
  errorMsg={errors?.countries?.message}
/>
```
