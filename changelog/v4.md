## [4.0.0](https://github.com/nishkohli96/rhf-mui-components/tree/v4.0.0)

**Released - 11 May, 2025**

- Change packageManager from `yarn` to `pnpm`
- Update `setup.sh` script.
- Add Desktop, Mobile and Static date and time picker components in `@mui-pickers` module.
- `shouldDisableOption` added to `RHFRadioGroup`, `RHFSelect`, `RHFNativeSelect`
- Add `aria-checked` and `aria-disabled` attributes to checkbox and radiobuttons
- Add `uploadedFiles` prop to `RHFFileUploader`, `validateFileList` now accepts `FileList | File[]` as input, add optional drag and drop functionality.
